# Ranks Tutorial - by Dvaelin
Requirements:
- Oraxen Plugin (Must Have!) | https://www.spigotmc.org/resources/%E2%80%8D%E2%9C%85-25-%E2%98%84%EF%B8%8F-oraxen-add-items-blocks-armors-hats-food-furnitures-plants-and-gui.72448/
- LuckPerms Plugin | https://luckperms.net/
- LPC - Chat Formatter Plugin | https://www.spigotmc.org/resources/lpc-chat-formatter.68965/
- TAB Plugin (Free or Paid) | https://www.mc-market.org/resources/20631/
- RealScoreboard Plugin (Recommended) | https://www.spigotmc.org/resources/realscoreboard-1-13-to-1-18.22928/

How to?
1. Extract Files

2. Copy or Move the png file inside "Ranks" folder to yourserver/plugins/oraxen/pack/textures/default/chat

3. Then copy the "chat_tags.yml" to yourserver/plugins/oraxen/glyphs 
   or copy the text inside to your yourserver/plugins/oraxen/glyphs/chat_tags.yml

4. And reload the configuration inside the game or console with this commands:
   /oraxen reload all or /oraxen reload pack (i recommend reload all)

# Item Rarity Tutorial - by Dvaelin
Requirements:
- Oraxen Plugin (Must Have!) | https://www.spigotmc.org/resources/%E2%80%8D%E2%9C%85-25-%E2%98%84%EF%B8%8F-oraxen-add-items-blocks-armors-hats-food-furnitures-plants-and-gui.72448/

Hot to?
1. Extract Files

2. Copy or Move the png file inside "Item Rarity" folder to yourserver/plugins/oraxen/pack/textures/default/chat

3. Then copy the "chat_tags.yml" it's the same like ranks to yourserver/plugins/oraxen/glyphs 
   or copy the text inside to your yourserver/plugins/oraxen/glyphs/chat_tags.yml

4. Go to oraxen/items/weapons.yml (this is just an example), then just add <glyph:common_tag> to the lore, you can change it to uncommon,
   rare, ultrarare, epic, hero, or legend as you like. or you can check the example at "weapons.yml" inside the Item Rarity folder

5. And reload the configuration inside the game or console with this commands:
   /oraxen reload all or /oraxen reload pack (i recommend reload all)
